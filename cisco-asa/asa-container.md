# Cisco Containerized ASAc Firewall

There is a Virtual ASA Firewall that is installable on Catalyst 9300 switches. Hardware requirements must be met. ASA Virtual Licensing is required.

* [Introduction ASAc Firewall on Catalyst 9300 Series][1] (Cisco Networking Blog)
  * Cisco Networking Blog post from December 2023.
 
## Cisco Modeling Labs

Cisco Modeling Labs (CML) includes a distribution of Cisco Images for simulating Routers, Switches, Firewalls, etc.
* The latest releaset of CML is based on Virtualization and Not Containerization
* The images provided for router/switch/firewall simulation are CML-compatible Virtual Machines
* In the future, maybe Cisco will make CML lighter weight and support container-based router/switch/firewall simulation?
